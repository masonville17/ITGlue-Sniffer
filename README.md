# ITGlue-Sniffer
Data Scraping for ITGlue API

##### **A simple data-scraping script**

---

## instructions:

If using ITG endpoint outside of USA, be sure to set endpoint URL variable, $ITGAPIEndpoint

Invoke with powershell 7+

It will take 5-6 minutes or longer, depending on size of your ITG tenant.

All available scraped data will be available in ./ITGlueExports folder as json
